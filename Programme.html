Programme de la Licence Informatique

Le programme s'organise en semestres et combine enseignements fondamentaux, enseignements transversaux (anglais, projet professionnel de l'étudiant...) et projets. Voici quelques-uns des enseignements phares de la formation :

  • Algorithmique et programmation Python (semestre 1 et 2) : ce module initie les étudiants aux bases de l'algorithmique et à la programmation Python : structures de contrôle, fonction, listes, dictionnaires et manipulation de fichiers. Il constitue le socle de tous les enseignements de programmation de la licence.

    En savoir plus sur Python
  • Introduction aux bases de données(semestre 1) : ce cours présente la modélisation entité-association, l'algèbre relationnelle ainsi que le langage SQL pour interroger et mettre à jour des données.

    En savoir plus sur SQL
  • Introduction au Web et Interface Homme-Machine(semestre 2) : ce module pose les bases du développement web (structuration de pages avec HTML, accessibilité) ainsi que les grands principes de conception d'interfaces centrées sur l'utilisateur.

    En savoir plus sur HTML
  • Réseaux (semestres 1, 2 et 5) : ces enseignements couvrent l'architecture des réseaux informatiques, les modèles OSI/TCP-IP, l'adressage, le routage ainsi que l'administration de base d'équipements réseau.

    En savoir plus sur le Réseaux
  • Programmation orientée objets en Java (semestre 3) : introduction à la programmation orientée objet à travers le langage Java : classes, héritage, polymorphisme et bonnes pratiques de conception logicielle.

    En savoir plus sur Java
